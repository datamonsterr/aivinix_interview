# Project skills

- `use-mise`: Use when editing `mise.toml` or environment pinning.
- `use-pnpm`: Use when editing `package.json`, lockfiles, or package install/run flows.
- `use-typescript`: Use when editing `tsconfig.json` or TypeScript source.
- `use-vitest`: Use when writing unit/integration tests.
- `use-nextjs`: Use when editing App Router routes, server components, or route handlers.
- `use-vercel`: Use when preparing deployment/runtime notes for Vercel.
- `use-agent-browser`: Use when browser automation/e2e flows need visual verification.
- `use-docker-compose`: Use when editing `docker-compose.yml` or container orchestration.
